# Mobile app (Flutter)

This folder contains a minimal Flutter code skeleton. Use it inside a Flutter project.

## How to use

1) Create a Flutter app in another folder:

flutter create event_access_mobile

2) Copy the files from this folder into:

event_access_mobile/lib

3) Add dependencies in pubspec.yaml:

http: ^1.2.2
